# 西安交通大学用户信息查询

## 安装

1. 解压到服务器，根目录设置到`public/`文件夹

2. `composer install`

3. 把`app/config.example.php`复制到`app/config.php`，修改其中的配置信息

    出于保密原因，其中的闭包验证的具体写法不能透露
